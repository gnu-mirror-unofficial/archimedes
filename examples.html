Archimedes

Examples

Simulated Devices
In this page, you can find many examples simulated by means of Archimedes. These examples are reported in order to show the advanced features of this simulator.

Each example displays the Input Files for Archimedes that is used to obtain these results.

In this page you can find two sections. The first shows Archimedes and the second shows several calculated IV curves.
