Contacts and Boundaries

In this section, we describe the various contacts we can choose in GNU Archimedes. First of all, we can divide the contacts into three categories: Insulator boundary, Ohmic and Schottky contacts. In this code we immagine a contact, or a boundary, as a line on an edge of the device. In GNU Archimedes there is no limit in the number of contacts, so you can specify whatever contacts you want.
