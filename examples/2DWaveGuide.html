2D Quantum Wave Guide

Quantum Archimedes is able to simulate the well-known Schroedinger-Poisson system along with open/closed consistent quantum boundary conditions. The device is described by means of a ASCII input file along with a 2D triangular mesh for the device geometry. This means that the Schroedinger-Poisson system is solved by means of a finite element method and then able to simulate very general geometry devices.
